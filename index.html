Aplicações HTML desafio DIO

Este é meu primiro arquivo HTML e estou aplicando os conhecimentos adquiridos atravéz dos cursos DIO. Alguns desses conhecimentos são as tags, e como primeiro desafio, é proposto a navegação entre paginas.

Algumas das tags utilizadas serão listadas abaixo

  1. strong - Para deixar fontes em negrito
  2. i - Para deixar fontes em itálico
  3. h1 - Utilizado para titulos
  4. h2 - Utilizado para subtitulos
  5. a - Ancoragens de links, fazendo com que seja possivel a navegação entre paginas
  6. ol - Para criar uma lista ordenada
